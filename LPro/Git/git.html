Курс по гитхабу. Краткое пособие по командной строке, то бишь ключевой функционал.


Глава 0. Гитхаб сайт.

Создать репозиторий, инициализировать ридми и гинигнором. Гитигнор компилировать в сервисе.

Глава 1. В строке

git --version данная команда отображает версию, проверка факта установки.

git status показывает подсказки, по состоянию на выбранную папку. Наличие модификаций, создание комита, отправка.

Глава 2. В декстоп прог

Возобновление работы не исследовано. Мой прогресс сохранён от старта использования.

Сначала лучше создать репозиторий на сайте, и инициализировать. Затем в приложении клонировать его в пустую папку, и докачать туда нужные файлы.

После изменения содержимого папки, в нижнем левом углу программы можно собрать изменения в коммит. Храниться он будет локально. В верхней строке определится кнопка Push origin, которая пересылает коммиты в гитхаб.

Ветки. Создавать, пулреквестить на сайте, фетчить мастера в приложении.
