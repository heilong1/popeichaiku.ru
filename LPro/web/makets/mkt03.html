Меню:
  • Первый конспект.
  • CSS-конспект.
  • Второй макет.
  • Четвёртый макет.

#cnt {
margin: 10px;
border: 2px green outset;
text-align: justify;}

#cnt #nav { //дочерний селектор
float: right;
margin-left: 10px; //"добавлено поле"
border: 2px blue solid;
width: 300px;}

Здесь немного сложнее. Дочерний селектор использован, чтобы наследовать все стили. При этом, кроме наследования, для корректной работы необходимо использовать вложение разделов в теле.

<div id="cnt">
<div id="nav"> </div>
</div>

Несмотря на плавающее размещение раздела nav, граница основного раздела cnt охватила всю ширину страницы. Важно, что текст продолжает обходить раздел. Границы вряд ли будут нужны при создании настоящего сайта, потому здесь можно игнорировать отсутствие миловидности. За изображением граница пропадает, но над текстом меню была. Вероятно, стоило задать белый фон, он должен убирать прозрачность. Чтобы текст немного отодвинулся от меню, "добавлено поле".
