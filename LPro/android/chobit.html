Подготовка.

Для начала потребовалось скачивать некую Java от Oracle. Java SE Development Kit 8u91 for Win64. Как-то так. Java SE Runtime Environment 8u91 где-то в комплекте должен быть, или ставить его тоже нужно. Скачайте и запустите установщик, убедитесь, что ‘Development Tools’, ‘Source Code’ и ‘Public JRE’ включены в установку в окне ручной установки.
